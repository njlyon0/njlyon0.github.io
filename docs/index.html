Welcome!

I am an early career data scientist with experience managing ecological data in a range of formats. The bulk of my coding experience is in R (including R Shiny) but I also have moderate competency with SQL and the command line. I plan on learning more coding languages going forward as data challenges necessitate (I am particularly excited to learn Python). I firmly believe in transparency in data management and am an avid git user. Check out my GitHub profile for examples of my ongoing and past work!
